# AI健康管家 - 功能模块说明

## 应用概述
AI健康管家是一个全方位的智能健康服务平台,集成了问诊、用药、运动、医美等多个健康管理模块,为用户提供专业的健康咨询和商品推荐服务。

## 核心功能模块

### 1. 核心医疗模块

#### 1.1 AI智能问诊 (Chat页面)
- **核心功能**: 基于用户输入的症状分析可能疾病,提供专业诊疗建议
- **AI能力**: 接入文心大模型,支持智能对话和症状分析
- **用户场景**: 用户出现感冒、肠胃不适等症状时,在线咨询病因及治疗方案
- **关联商品**: 自动识别药品关键词,提供购买弹窗,支持跳转淘宝购买
- **特色功能**: 
  - 流式对话,实时响应
  - 药品关键词智能识别
  - 处方药安全提醒
  - 快速提问模板

#### 1.2 药品检索 (Search页面)
- **核心功能**: 支持药品功效、用法用量查询;AI智能总结药品信息
- **AI能力**: 接入文心大模型,生成药品使用总结
- **用户场景**: 用户需购买降压药时,查询药品信息和使用方法
- **关联商品**: 各类药品、药用辅料、智能药盒
- **特色功能**:
  - AI智能总结
  - 快速搜索建议
  - 药品信息详细展示

### 2. 运动健康模块 (Sports页面)

#### 2.1 运动方案定制
- **核心功能**: 根据用户体重、运动史等身体数据,生成个性化健身计划
- **AI能力**: 接入文心大模型,提供专业运动指导
- **用户场景**: 用户想减脂时,获取适合自己的运动计划
- **关联商品**: 瑜伽垫、哑铃、运动服饰、蛋白粉
- **特色功能**:
  - 个性化运动方案
  - 运动强度指导
  - 居家/健身房场景支持

#### 2.2 运动损伤咨询
- **核心功能**: 分析运动损伤症状,提供康复建议及护理方案
- **AI能力**: 接入文心大模型,提供专业康复指导
- **用户场景**: 用户跑步后膝盖疼痛,咨询损伤原因及恢复方法
- **关联商品**: 运动护具(护膝、肌贴)、筋膜枪、理疗产品
- **特色功能**:
  - 损伤症状分析
  - 康复方案推荐
  - 护具产品推荐

### 3. 医美模块 (Beauty页面)

#### 3.1 医美项目咨询
- **核心功能**: 评估医美项目适用性,提示潜在风险,推荐合规医美机构
- **AI能力**: 接入文心大模型,提供专业医美咨询
- **用户场景**: 用户想做光子嫩肤,咨询自身肤质是否适合及术后护理要点
- **关联商品**: 医美服务套餐、术后修复产品、家用美容仪
- **特色功能**:
  - 项目适用性评估
  - 风险提示
  - 术后护理指导

#### 3.2 医美产品推荐
- **核心功能**: 筛查医美产品合规性,根据用户肤质推荐护肤品
- **AI能力**: 接入文心大模型,提供产品成分分析
- **用户场景**: 用户需选购合规玻尿酸原液,咨询产品成分与效果
- **关联商品**: 医美护肤品、医用耗材、合规医美仪器
- **特色功能**:
  - 产品合规性筛查
  - 成分分析
  - 肤质匹配推荐

### 4. 健康知识服务模块

#### 4.1 疾病科普 (Diseases页面)
- **核心功能**: 提供常见疾病的基础知识介绍和健康科普内容
- **用户场景**: 用户想了解感冒、高血压等常见疾病的预防和治疗
- **特色功能**:
  - 疾病分类浏览
  - 症状、预防、治疗详细说明
  - 相关药品推荐

#### 4.2 用药指南 (Guide页面)
- **核心功能**: 提供安全用药知识,特殊人群用药指导
- **用户场景**: 用户需了解儿童、孕妇、老年人用药注意事项
- **特色功能**:
  - 用药安全提示
  - 特殊人群用药指南
  - 常见用药误区

## 技术特点

### AI能力
- 所有问答模块均接入文心大模型
- 支持流式对话,实时响应
- 智能场景识别,提供专业建议
- 系统提示词优化,确保回答专业性

### 商品推荐
- 智能关键词识别
- 商品卡片展示
- 支持跳转淘宝购买
- 价格、评价、品牌综合推荐

### 用户体验
- 响应式设计,适配移动端
- 橙绿配色方案,清晰易读
- 图标与文字对比度高
- 快速提问模板,提升效率

### 安全合规
- 处方药安全提醒
- 医美项目风险提示
- 产品合规性筛查
- 专业免责声明

## 使用说明

1. **首页**: 浏览所有功能模块,快速进入所需服务
2. **AI问诊**: 描述症状,获取专业诊疗建议
3. **药品检索**: 搜索药品,查看详细信息和AI总结
4. **运动健康**: 获取运动方案或咨询运动损伤
5. **医美咨询**: 了解医美项目或选购医美产品
6. **疾病科普**: 学习疾病知识,做好预防
7. **用药指南**: 掌握安全用药知识

## 注意事项

⚠️ **重要提示**:
- 本应用提供的信息仅供参考,不能替代专业医生的诊断和治疗
- 处方药需咨询医生并获得处方后购买
- 医美项目需选择正规机构,充分了解风险
- 如有严重症状或疾病,请及时就医
