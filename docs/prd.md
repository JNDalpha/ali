# AI药师应用需求文档

## 1. 应用概述

### 1.1 应用名称
AI药师助手

### 1.2 应用描述
面向普通患者的智能用药指导应用，通过AI技术提供疾病科普、用药建议和药品推荐服务，集成在淘宝APP中，为用户提供便捷的健康咨询和购药体验。

## 2. 目标用户
普通患者群体，通过淘宝APP图标访问

## 3. 核心功能

### 3.1 疾病信息科普
提供常见疾病的基础知识介绍和健康科普内容

### 3.2 智能问答
基于AI技术的智能客服功能，能够理解用户描述的症状并提供相应回复，所有问答界面接入文心大模型进行问答

### 3.3 用药建议生成
- 根据用户描述的症状生成用药建议
- 涵盖处方药和非处方药建议
- 对于需要处方药治疗的病症，提醒用户咨询专业医生并获得医生处方后购买
- 提供在线咨询医生的跳转窗口

### 3.4 药品检索
支持用户主动搜索和查询药品信息

### 3.5 药品推荐榜单
从同一种药品对应的大量商品中，结合以下因素进行智能推荐：
- 购买量
- 好评率
- 品牌知名度
- 性价比

### 3.6 药品链接跳转
在药品推荐处提供可跳转的链接，直接跳转到阿里大药房电商平台

### 3.7 精准配送
根据用户定位提供精准快速的药品送达服务

## 4. 平台集成
应用以图标形式内置在淘宝APP中，用户可直接通过淘宝APP访问

## 5. 图标设计要求

### 5.1 颜色方案
图标设计以橙色为主色调，绿色为辅助色

### 5.2 显示要求
- 修复图标无法正常显示的问题
- 确保图标与文字清晰可分辨
- 保证在不同设备和分辨率下的显示效果

## 6. 技术集成
所有问答界面接入文心大模型，提供更智能准确的问答服务

## 7. 安全提醒
对于涉及处方药的用药建议，系统将明确提醒用户需咨询专业医生并获得处方后购买，确保用药安全
